Security Analytics Dashboard for Logsign SIEM

A modern web application for analyzing security logs, network events, and authentication attempts exported from Logsign SIEM in HTML format.

The application automatically parses reports generated by Logsign SIEM and presents data using professional dashboards, hourly heatmaps, TOP 5 rankings, and detailed host/user analytics. The system helps detect anomalies, brute-force attempts, port scanning activity, denied connections, and suspicious network behavior.

Key Features
Logsign SIEM HTML report parsing
security event visualization
hourly activity heatmaps
TOP 5 users, hosts, ports, and services
internal and external port scanning analysis
failed login attempt analysis
denied network connection analysis
detailed host and user investigation panels
modern SOC / Blue Team styled dashboard
Benefits
faster security incident detection
simplified analysis of large SIEM log datasets
automation of Logsign report processing
clear visualization of threats and activity
support for administrators, SOC teams, and cybersecurity analysts
reduced incident investigation time

The project was built using PHP + TailwindCSS as a lightweight and high-performance analytics platform for IT security environments.



Nowoczesna aplikacja webowa do analizy logów bezpieczeństwa, zdarzeń sieciowych oraz prób logowania eksportowanych z systemu Logsign SIEM w formacie HTML.

Aplikacja automatycznie parsuje raporty generowane przez Logsign SIEM i prezentuje dane w postaci profesjonalnych dashboardów, heatmap godzinowych, rankingów TOP 5 oraz szczegółowych analiz hostów i użytkowników. System umożliwia szybkie wykrywanie anomalii, prób brute-force, skanowania portów, odrzuconych połączeń oraz podejrzanej aktywności sieciowej.
